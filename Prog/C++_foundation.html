# 基本框架

在学习C++之前，我们需要了解到，所有的高级语言都有自己的基本框架，是基础中的基础，不用管其他，先背下来就好

#include <iostream>
using namespace std;

int main() {
  
  return 0;
}

/*
需要使用的函数与规则

主函数，必定运行的地方
*/

[!NOTE]

# 例子解释

第 1 行：#include <iostream> 是一个头文件库，允许我们使用输入和输出对象，如第 5 行中使用的 cout。头文件为 C++ 程序增加了功能。

第 2 行：using namespace std 表示我们可以使用标准库中的对象和变量的名称。

如果不理解 #include <iostream> 和 using namespace std 是如何工作的，也不用担心。只要认为它们是（几乎）总是出现在您的程序中的东西。

**第 3 行：**一个空行。C++ 会忽略空白。但我们用它来使代码更具可读性。

**第 4 行：**C++ 程序中总是出现的另一件事是 int main()。这被称为一个函数。其大括号 {} 内的任何代码都将被执行。

第 6 行：return 0; 结束主函数。

**注意：**每个 C++ 语句都以分号 ; 结尾。

注意：int main() 的主体也可以写成：int main () { return 0; }

请记住：编译器会忽略空白。但是，多行使代码更具可读性。

**第 7 行：**不要忘记添加右花括号 } 以真正结束 main 函数。

# 输出与输入

# C++ 输出（cout）

cout 对象与 << 运算符一起用于输出值/打印文本：

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World!";
  return 0;
}
/*
运行结果:
Hello World!
*/

也可以根据需要添加任意数量的 cout 对象。但是，请注意，它不会在输出末尾插入新行：

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World!";
  cout << "I am learning C++";
  return 0;
}
/*
运行结果:
Hello World!I am learning C++
*/

换行

要插入新行，您可以使用 \n 字符：

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World! \n";
  cout << "I am learning C++";
  return 0;
}

还可以使用另一个 << 运算符，并将 \n 字符放在文本后面，如下所示：

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World!" << "\n";
  cout << "I am learning C++";
  return 0;
}

**提示：**两个连续的 \n 字符将创建一个空行：

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World! \n\n";
  cout << "I am learning C++";
  return 0;
}

另一种插入新行的方法是使用 endl 操纵符：

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World!" << endl;
  cout << "I am learning C++";
  return 0;
}

\n 和 endl 都用于换行。然而，\n 最常用。

换行符（\n）被称为转义序列，它强制光标更改其在屏幕上的位置到下一行的开头。这就产生了一个新行。

其他有效的转义序列实例有：

转义序列 描述
\t 水平制表符（即 Tab 键的效果）
\\ 插入反斜杠字符 ()。
\" 插入双引号字符。

# 分支

# 判断

if：如果

#include <iostream>
using namespace std;

int main()
{
	int a = 10;
	if (a == 10)
		cout << "a是10";
	return 0;
}

else：否则

需要紧跟if或else if

#include <iostream>
using namespace std;

int main()
{
	int a = 666;
	if (a == 10)
		cout << "a是10";
    //cout << "这是分支结构"; 在这个地方会打断   
    else
        cout << "a不是10";
	return 0;
}

else if：否则如果

#include <iostream>
using namespace std;

int main()
{
	int a = 666;
	if (a == 10)
		cout << "a是10";
    else if(a == 666)
        cout << "a是666";
    else
        cout << "a不是10";
	return 0;
}

总结

/*
if (条件一) 开头
    事件一
else if(条件二) 身体
    事件二
else 尾巴
    事件三
*/

# 选择

# 基本结构

#include <iostream>
using namespace std;

int main()
{
	int a = 0;
	switch (a)
	{
	case 0:
		cout<<"a等于0"<<endl;
		break;
	case 1:
		cout<<"a等于1"<<endl;
		break;
	default:
		cout<<"a不等于上面的所有"<<endl;
		break;
	}
	return 0;
}

# 总结

/*
switch (变量)
{
case 值一:
    事件一
    break;
case 值二:
    事件二
    break;
default:
    事件三
    break;
}
*/

# 循环

# 次数

#include <iostream>
using namespace std;

int main()
{
	for (int i = 0; i < 10; i++)
	{
		cout<<"i是"<<i<<endl;
	}
	return 0;
}

for(变量的起始值；条件；变量的每次变化值){
   条件为真时执行，条件为假跳出循环
}

# 条件

#include <iostream>
using namespace std;

int main()
{
    int i=0;
    while(i<10)
    {
        cout<<i<<endl;
        i++;
    }
	return 0;
}

while(条件)
{
    条件为真时执行，条件为假跳出循环
}

# 跳出循环

break：跳出所有循环，常与if搭配使用

#include <iostream>
using namespace std;

int main()
{
    int i=0;
    while(i<10)
    {
        cout<<i<<endl;
        i++;
        if (i==6){
            break;
        }
    }
	return 0;
}

continue：跳出当次循环

#include <iostream>
using namespace std;

int main()
{
    int i=0;
    while(i<10)
    {
        cout<<i<<endl;
        i++;
        if (i==6){
            continue;
        }
    }
	return 0;
}

# 2、变量定义域

解释：变量生效的范围

例2.1

#include <iostream>
using namespace std;

void a(){
    int b = 10；
}

int main(){
    a();
    cout<<b;
    return 0;
}

[!CAUTION]

无法编译通过，在 cout<<b;报错

原因：b的定义在a()这个子函数内，故main()内无b的定义；

例2.2

#include <iostream>
using namespace std;

int b = 10;

void a(){    
    cout<<b;
}

int main(){
    a();
    cout<<b;
    return 0;
}

/*
无报错，输出1010
*/

例2.3

#include <iostream>
using namespace std;

int b = 10;

void a(){  
    b = 5;
    cout<<b;
}

int main(){
    a();
    cout<<b;
    return 0;
}

[!NOTE]

我们称b为全局变量，每一个函数对其修改都生效

例2.4

#include <iostream>
using namespace std;

int b = 10;

void a(){  
    int b = 5;
    cout<<b;
}

int main(){
    a();
    cout<<b;
    return 0;
}

[!NOTE]

我们在a()内重新定义了b，对全局变量不产生影响

# 指针

建立传送门，直接指向某个变量

#include <iostream>
using namespace std;

int main()
{
    string food = "Pizza";

    cout << food << endl;  // 输出变量信息
    cout << &food << endl; // 输出变量地址
    return 0;
}

#include <iostream>
using namespace std;

int main()
{
    string food = "Pizza";  // 变量声明
    string*
        .ptr = &food;    // 指针声明

    // 引用：使用指针输出 food 的内存地址 (0x6dfed4)
    cout << ptr << "\n";

    // 解引用：使用指针输出 food 的值 (Pizza)
    cout << *ptr << "\n";
    return 0;
}
