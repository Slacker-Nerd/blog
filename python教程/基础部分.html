# 输入输出

在理解 “输入输出” 的核心概念前，我们不妨先想一个关键问题：为什么编程中必须有输入输出？

本质上，编程的核心是让计算机执行运算、逻辑判断与数据处理 —— 而这一切操作，都发生在计算机的内存之中。内存就像一个 “临时工作台”，所有计算过程和中间结果都在这里完成，但问题也随之而来：内存中的数据是临时的，既无法被外部的使用者直接看见，也无法接收来自使用者的指令或数据。

为了解决 “人机交互” 的这一核心矛盾，“输入输出”（简称 I/O）的概念应运而生。它就像连接人与计算机内存的 “桥梁”：输入，是我们将指令、数据传递给计算机的方式（比如键盘输入、文件导入）；输出，则是计算机将处理结果反馈给我们的途径（比如屏幕显示、生成报表）。没有输入输出，再强大的计算能力也只是内存中无人知晓的 “沉默运算”，无法真正服务于实际需求。

在python中，输入指令其实非常简单——input()，而输出在编程语言中又叫打印，顾名思义他的指令就是——print()。

在知道这两个函数之后，我们就可以写一个最简单的程序

a=input("你想让我说什么")
print("电子摆烂仔让我告诉全世界"+a)

运行完上面的代码，我们就能看到一个最基本的效果

你想让我说什么:我想发大财
电子摆烂仔让我告诉全世界他想：我想发大财

当然，在这个代码里也出现了一个我们刚才没有讲过的东西——a,这个也就是我们下一个部分要学习的东西变量

# 注解

代码注解（也叫注释）是嵌入在代码中的说明文字，Python 解释器会直接忽略它们，不会执行。但对程序员来说，注释是理解代码的 “导航灯”—— 它能解释代码的功能、思路或注意事项，既方便自己日后回看，也便于他人协作。

1. 单行注释（#）

用井号#开头，注释内容从#开始到行尾结束，适用于简短说明。

# 计算圆的面积（这是单行注释）
radius = 5  # 定义圆的半径（行内注释，说明变量用途）
area = 3.14 * radius * radius
print(area)  # 输出计算结果

2.多行注释（三引号）

用三单引号''' '''或三双引号""" """包裹，适用于长篇幅的注释（如函数说明、模块介绍）。

"""
这是一个计算长方形面积的程序
作者：xxx
日期：2024年
"""

def calculate_area(length, width):
    '''
    函数功能：计算长方形面积
    参数：
        length: 长方形的长
        width: 长方形的宽
    返回值：
        长方形的面积
    '''
    return length * width

3. 文档字符串（Docstring）

属于特殊的多行注释，用于描述模块、函数、类的功能，可通过.__doc__属性查看，是 Python 的 “官方注释规范”。

def add(a, b):
    """
    实现两个数的加法运算
    
    参数：
        a (int/float): 第一个加数
        b (int/float): 第二个加数
    
    返回：
        int/float: 两个数的和
    """
    return a + b

# 查看文档字符串
print(add.__doc__)

# 变量

什么是变量？变量的的作用是什么？

在编程中，变量就像我们生活里的 “带标签储物柜”—— 它是一块被划定的计算机内存空间，专门用来存放程序运行过程中需要用到的数据（比如数字、文字、符号等），而 “变量名” 就是贴在这个柜子上的标签，让我们能快速找到并使用里面的内容。

就像之前代码里的a，当我们执行a = input("你想让我说什么：")时，其实是在计算机内存里开辟了一块空间，把你输入的 “我想发大财” 这个内容存进去，再给这块空间贴上个叫a的标签。之后用到a，本质就是调用这块空间里的内容。

变量的核心特点是 “可变化”—— 我们可以随时修改它存储的数据。比如先让a = 10，过一会儿又能让a = "hello"，就像给同一个储物柜换了里面的东西，标签却始终清晰。

# 变量的命名规则

  1. 由字母、数字和下划线组成

    变量名只能包含英文字母（a-z, A-Z）、数字（0-9）和下划线（_），不能包含空格、中文或其他特殊符号（如!@#$%^&*()等）。

    • ✅ 正确：user_name, age18, _score
    • ❌ 错误：user name, 年龄, score!, a+b

  2. 不能以数字开头

    变量名可以以字母或下划线开头，但绝对不能以数字开头。

    • ✅ 正确：name1, _123test
    • ❌ 错误：1name, 2024_report

  3. 区分大小写

    Python 对大小写敏感，Age、age和AGE会被视为三个完全不同的变量。

  4. 不能使用 Python 关键字

    关键字是 Python 语言本身预留的特殊单词（如if、else、for、while、def、class等），它们有特定的语法功能，不能用作变量名。

    • ✅ 正确：my_if, class_name
    • ❌ 错误：if, class, True, None

示例：规范的变量命名

# 正确且规范的命名
user_name = "张三"
student_age = 20
total_score = 95.5
is_pass = True

# 错误的命名
1age = 20  # 以数字开头
user name = "李四"  # 包含空格
class = "一年级"  # 使用关键字
$price = 99  # 包含特殊符号

# 变量的类型

整数（int）

没有小数部分的数字，如年龄、数量。

age = 18          # int类型
count = 100       # int类型

浮点数（float）

带有小数点的数字，如价格、身高。

price = 99.9      # float类型
height = 1.75     # float类型

布尔值（bool）

只有True（真）和False（假）两个值，用于逻辑判断。

is_student = True  # bool类型
has_money = False  # bool类型

字符串（str）

用于表示文本，用单引号' '、双引号" "或三引号''' '''/""" """包裹。

name = "张三"                # 双引号字符串
message = 'Hello Python'     # 单引号字符串
paragraph = """这是一段
多行文本"""                 # 三引号字符串（支持换行）

# 变量的使用方法

1、定义

所有的变量在使用时，都需要定义，但python在定义方法有点特殊，我们只需要直接给他赋予值即可，一般有以下几种方法

a = 10 # 赋予一个具体的值
b = input() # 等待用户输入
c = None # 设为空，在后期赋值时避免发生错误，前期基本不会用到

2、修改

在修改时，我们要先知道一个概念，类型匹配，例如在下面的这种情况

  • 使用input()时，它的返回值是字符串，在加法时就会出现以下情况

    a = input() # 6
b = input() # 66
print(a+b) # 666

# 运算符

需要注意所有经过运算符处理的结果是存在电脑的内存空间里的，具体的可以参考以下代码

a=10
a+10 # 此时的运算结果是20，但a还是10
print(a) #10
a=a+10 # 此时的运算结果是20，并且将这个结果赋予了a,a此时的值20
print(a) #20

1.算术运算符：用于执行数学运算。

运算符 描述 示例 返回类型
+ 加法 a + b int 或 float
- 减法 a - b int 或 float
* 乘法 a * b int 或 float
/ 除法 a / b float
// 整除 a // b int
% 取余 a % b int
** 幂运算 a ** b int 或 float

2.比较运算符：用于比较两个值，返回布尔值（True 或 False）。

运算符 描述 示例 返回类型
== 等于 a == b bool
!= 不等于 a != b bool
> 大于 a > b bool
< 小于 a < b bool
>= 大于等于 a >= b bool
<= 小于等于 a <= b bool

3. 逻辑运算符：用于组合布尔表达式的运算符。

运算符 描述 示例 返回类型
and 逻辑与 a and b bool
or 逻辑或 a or b bool
not 逻辑非 not a bool

4. 复合运算符：用于将值赋给变量。

运算符 描述 示例 等同于
= 赋值 a = 5
+= 加法赋值 a += 5 a=a+5
-= 减法赋值 a -= 5 a=a-5
*= 乘法赋值 a *= 5 a=a*5
/= 除法赋值 a /= 5 a=a/5

5. 身份运算符 ：用于比较两个对象是否是同一个

运算符 描述 示例 返回类型

运算符 描述 示例 返回类型
